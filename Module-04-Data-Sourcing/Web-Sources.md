## 🛠️ 2. Core Library Documentation
The essential manuals for the "Scraper's Toolkit":
* **[BeautifulSoup4](https://www.crummy.com/software/BeautifulSoup/bs4/doc/):** HTML/XML parsing.
* **[Playwright Python](https://playwright.dev/python/docs/intro):** Headless browser automation.
* **[HTTPX](https://www.python-httpx.org/):** Next-generation async HTTP client.
* **[Tabula-py](https://tabula-py.readthedocs.io/):** PDF table extraction.
* **[Geopy (Nominatim)](https://geopy.readthedocs.io/):** Geocoding and spatial data.
