## 🔍 Detailed Weekly Breakdowns

### **Week 1-2: The Infrastructure Phase**
* **CLI Mastery:** Moving beyond the GUI to manage files and processes via the terminal.
* **Version Control:** Establishing a "Single Source of Truth" using Git branches and remote repositories.
* **API Architecture:** Understanding how modern backends (FastAPI) handle requests, validation, and execution constraints.

### **Week 3-4: The Intelligence & Sourcing Phase**
* **Semantic Search:** Using embeddings to find meaning in data rather than just keyword matches.
* **Data Mining:** Automating the collection of data from "messy" sources like public websites and locked PDFs.
* **Multimodal AI:** Learning to handle non-text data (Base64 encoding/decoding) and function calling for structured outputs.

### **Week 5-6: The Engineering & Analytics Phase**
* **Data Janitorial Work:** Using OpenRefine to solve the "inconsistent naming" problem and Shell pipelines for massive log files.
* **High-Performance SQL:** Leveraging **DuckDB** to run complex analytical queries on millions of rows without a server.
* **Spatial Intelligence:** Merging demographic data with geographic boundaries (Shapefiles) for Manhattan-scale business analysis.

### **Week 7: The Narrative Phase**
* **Data Storytelling:** Learning that a chart is a "So What?" tool. Combining numbers, text, and illustrations to influence decisions.
* **Reactive Notebooks:** Transitioning from static Jupyter cells to **Marimo’s** Directed Acyclic Graph (DAG) for reproducible and interactive apps.
* **Creative Viz:** Using **Comicgen** and Google Sheets to turn technical data clusters into human-centric comic narratives.
